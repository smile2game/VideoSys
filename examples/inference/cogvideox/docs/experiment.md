# 单卡加速和多卡并行实验 
### 2x3090
单卡实验：
| 模型/延迟    | base   | pab    |
| ------------ | ------ | ------ |
| cogvideoX-2b | 196.52 | 148.54 |
| 加速比       | 1      | 1.32   |

多卡实验：
| 模型/卡数    | 1      | 2      |
| ------------ | ------ | ------ |
| cogvideoX-2b | 196.52 | 128.32 |
| 加速比       | 1      | 1.53   |

note:pab和dsp并行能够叠加同时使用
